Cross-Domain Hyperspectral Image Classification
Zhiyu Jiang, Jianing Li, Shijie Xu, Zhuozhao Liu, Dandan Ma*, Qi Wang, and Yuan Yuan. Pattern Recognition, 168, 111836, 2025.
[DOI] [PDF]

This work provides a systematic review and task framework for cross-domain hyperspectral image classification, organizing recent advances into label-limited, label-free, unknown-label, and sample-free settings. It further discusses spectral heterogeneity and cross-device classification, highlighting challenges and future directions for robust remote sensing perception.



Structural Consistency Learning for Unsupervised Domain Adaptive Object Detection
Zhiyu Jiang, Jie Chen, and Yuan Yuan*. Neural Networks, 191, 107767, 2025.
[DOI] [PDF]

This paper proposes a structural consistency learning framework for unsupervised domain adaptive object detection. The method introduces enhanced dual-attentional feature alignment and a structural feature consistency module to reduce background interference, address category imbalance, and improve cross-domain detection performance.



Implicit CLIP Prior Decoupling for Few-Shot Remote Sensing Image Segmentation
Zhiyu Jiang, Ye Yuan, Dandan Ma*, Qi Wang, and Yuan Yuan. IEEE Transactions on Geoscience and Remote Sensing, 63, 2025.
[DOI] [PDF] [Code]

ICPD-Net mines implicit vision-language priors from RemoteCLIP for few-shot remote sensing segmentation. By combining implicit prior decoupling, reliable feature fusion, and dual visual priors, the method mitigates intraclass variation and achieves state-of-the-art performance on standard remote sensing FSS benchmarks.



Multimodal Difference Augmentation Learning for Remote Sensing Change Detection
Hao Yang, Zhiyu Jiang*, Dandan Ma, and Qi Wang. IEEE Transactions on Geoscience and Remote Sensing, 63, 2025.
[DOI] [PDF] [Code]

MdaCD enhances remote sensing change detection by exploiting multimodal vision-language cues and difference augmentation learning. It introduces CLIP-guided masking and multimodal difference augmentation to suppress unchanged regions, strengthen change-aware representations, and improve change localization.
